# Phase 2-B 作業の進め方（串さん宛）

串さん

お疲れ様です。Phase 2-B の作業の進め方を共有します。
ルート直下の index.html で行った Phase 2-A と同じ手順で進めてください。

## 作業の流れ（1ページごとに繰り返し）

```
① Claude Code に Plan 作成を依頼
② Plan の内容を確認・承認
③ チケット単位で Claude Code に実行を依頼（まとめて渡さない）
④ コンパイル + ブラウザ確認（PC / SP）
```

## 串さんの担当ページ（10ページ）

price / protein / reservation / rules / sitemap / sportip / storelist / visitor / point_redemption / personal

## 注意点

- `CLAUDE.md` に作業手順・規約・注意点がすべて書いてあり、Claude Code が自動で読み込みます。**このファイルは編集しないでください**
- Plan は丸ごと実行させず、必ず1チケットずつ依頼してください（スコープが大きいと整合性が崩れます）
- 共通コンポーネント（ヘッダー・フッター等）は Phase 1 で対応済みなので触らないでください

## 作業漏れを防ぐコツ

Plan 作成後と作業完了後に、以下のプロンプトを打つと Claude Code が自己チェックしてくれます：

「年収3000万円のフロントエンドエンジニアとして、客観的な視点で、先方フィードバックに記述されている根本原因が解消された内容になっていますか」

不明点あればいつでも聞いてください！
